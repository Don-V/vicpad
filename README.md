# VicPad

VicPad is a simple terminal text-editor built in C++ for my personal use and maybe yours

## Setup
Requires `gcc` and `ncurses`

Run the `Makefile`:

    $ make

## Usage
Open the application:
- create a new file:

        $ ./bin/vicpad

- open existing file:

        $ ./bin/vicpad already_existing_file.txt
